# Palny

Использовались:
- [Django](https://www.djangoproject.com/)
- [Pipenv](https://pypi.org/project/pipenv/)

## Локальный запуск
```
$ git clone https://github.com/Umatriz/palny.git
$ cd palny
$ pipenv shell
$ pipenv install --ignore-pipfile
$ python manage.py migrate
```
